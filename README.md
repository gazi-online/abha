# ABHA Card PVC Cutter

ABHA Card PDF থেকে PVC-ready image তৈরি করার tool।  
সম্পূর্ণ browser-side — কোনো data server-এ যায় না।

## Features
- PDF আপলোড করুন → Front + Back আলাদা হয়ে যাবে
- 300 DPI, 8.6cm × 5.4cm (CR80 PVC card size)
- Card border যোগ করুন — thickness ও color live preview
- Combined + individual image download

## Deploy on Vercel
1. এই repo টি GitHub-এ push করুন
2. [vercel.com](https://vercel.com) এ গিয়ে **New Project** → GitHub repo select করুন
3. Framework: **Other** (no build step needed)
4. **Deploy** চাপুন — ব্যস!
